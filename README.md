# Masterarbeit-Strassennamen
Dieses Repository enthält die Daten und Skripte für der Masterarbeit "Straßennamen als Konstruktion des kollektiven Gedächtnisses am Beispiel Dresdens".
</br></br>
## Aufbau
### Daten
Alle verwendete Daten befinden sich im Ordner 'data'. Die semi-automatisiert, semi-manuell erarbeitete Datenbank, auf der die Karten und Diagramme basieren, bedindet sich im Unterordner 'clean'.
### Skripte
Alle verwendeten Skripte befinden sich im Ordner 'scripts'. Die Skripte, die die Diagramme und Karten erzeugt haben, können im Unterordner 'analysis' eingesehen werden.
### Ergebnisse
Die erzeugten Karten können im Ordner 'results' eingesehen und heruntergeladen werden. Jede Karte liegt sowohl als statisches Bild in 'results/images' als auch als interaktive HTML-Version in "results/maps' vor.
Die erzeugten Diagramme befinden sich im Ordner 'results/charts'. Einfache Diagramme sind in im Ordner 'results/charts/bar_charts' zu finden. Diagramme, welche verschiedene Stadtteile vergleichen im Ordner "results/charts/bar_charts_district_comparison'. Die Boxplot-Diagramme der Distanzanalyse können im Ordner 'results/charts/boxplots' eingesehen werde.
